# Claude Code 配置助手

快速配置 Claude Code 开发环境的文档集合。

---

## 使用配置助手

克隆项目后，进入目录并启动 Claude：

```bash
cd /path/to/cc-helper
claude
```

然后告诉 Claude：**"帮我配置"**

Claude 会读取 `CLAUDE.md` 并引导你选择需要配置的项目。

---

## 快捷键

| 快捷键 | 说明 |
|--------|------|
| `Shift + Tab` | 切换权限模式 |
| `Shift + Enter` | 换行（部分终端不支持时使用 `\ + Enter`） |

## 斜杠命令

| 命令 | 说明 |
|------|------|
| `/init` | 初始化项目 CLAUDE.md |
| `/terminal-setup` | 配置终端集成 |
| `/resume` | 选择历史会话 |
| `/rename` | 重命名当前对话 |
