## 国内网络加速

如果你在中国大陆，安装工具时可能遇到网络问题：

### npm 镜像

```bash
# 临时使用
npm install -g <package> --registry=https://registry.npmmirror.com

# 永久配置
npm config set registry https://registry.npmmirror.com
```

### GitHub 文件加速

```bash
# 使用 ghproxy 加速
https://ghproxy.com/https://raw.githubusercontent.com/...
```
